01 / THE FIVE SKILLS
001 fable-mode The full operating character: judgment, planning, orchestration, verification, output discipline. Stays on for the session. SAY "FABLE MODE" 002 fable-grill Structured facilitation: the staged interrogation, the war-game, the fork batch. Always ends in a decision ledger on disk. SAY "GRILL ME" 003 fable-spec The two-tier spec pipeline: intake grill → complete design spec → review package for your smartest model → ingest its replacement spec. SAY "SPEC THIS OUT" 004 fable-plan A staged plan-of-record a cold session can execute: walking skeleton first, every step carries a ground-truth Verify and a scope Fence. SAY "PLAN THIS BUILD" 005 fable-review A verdict backed by live probes run this session: claim ladder, adversarial refutation, explicit confidence marks. Never a vibe read. SAY "IS THIS ACTUALLY DONE"

Your daily model runs the skills. Your smartest model gets a paper, never a skill.

The loop for a big idea: you and your daily model grill and draft the spec. Your smartest model reviews it and returns one complete replacement. Your daily model ingests, plans, and builds, and fable-review gates the ship. Only one model? Run the review in a fresh session at high effort and say who reviewed it. Never skip the gate silently.

02 / INSTALL

Two commands.

With Claude Code installed:
claude plugin marketplace add uhhexe/fable
claude plugin install fable@fable
Then start a new session and say "fable mode" to check it's live. Update later with claude plugin update fable@fable. Inside a session, /plugin marketplace add uhhexe/fable works too.

Never build before the gate.
